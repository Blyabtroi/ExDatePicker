ExDatePicker
============
ExDatePicker is an extended picker for iOS, which has the following regimes:

- full (day, month, year);
- [day only] [3];
- [year only] [4];
- [day and month] [5];
- [month and year] [6];
- [custom values] [7] (like UIPickerView).
